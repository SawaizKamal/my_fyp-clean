- `backend/requirements.txt` - Python dependencies

### Important Notes

1. **FFmpeg**: MoviePy requires ffmpeg, which is available on Render's Linux environment by default

2. **Video Storage**: Videos are stored in `backend/data/` which is **ephemeral** (cleared on redeploy). For production, consider using cloud storage (S3, Cloudinary, etc.)

3. **Whisper Model**: Uses "base" model (~200MB RAM). Already cached to avoid reloading.

4. **Database**: PostgreSQL is automatically configured. Free tier works, but upgrade to Starter ($7/month) for better performance.
